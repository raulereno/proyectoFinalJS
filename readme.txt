Funciones del proyecto:
*Agregar productos al carro, con un finalizar compra que lleva a informacion para enviar el productos
*Login de usuario, el usuario es "admin" y la contraseña "1234", que abre la opcion al administrador de agregar productos que funcionan como cualquier otro productos
*Posibilidad de quitar productos del carro mediante un boton de delete
*Llamada a JSON en la pagina wikidogs, que muestra caracteristicas de una raza

Cosas que no agregue:
*Stock de productos debido a que se rompia mucho a la hora de actualizar el stock en los objetos, en el dom y en el local storage

Recomendaciones finales:

*Realizar un CLEAR del local storage al terminar de corregir, debido a que si utilizaba un clear se borraria el producto nuevo que se agrega.
